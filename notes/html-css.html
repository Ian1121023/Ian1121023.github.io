HTML/CSS 基礎入門

📅 2024年1月20日 👁️ 閱讀時間：10 分鐘 🏷️ 前端開發

HTML 和 CSS 是網頁開發的基礎。HTML 負責網頁的結構和內容，CSS 負責網頁的樣式和外觀。讓我們從基礎開始學習！

什麼是 HTML？

HTML（HyperText Markup Language）是一種標記語言，用於建立網頁的結構。它使用各種標籤（tags）來定義內容的類型和結構。

基本的 HTML 結構

<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <title>網頁標題</title>
</head>
<body>
    <h1>這是標題</h1>
    <p>這是段落</p>
</body>
</html>
💡 提示：每個 HTML 文件都應該以 <!DOCTYPE html> 開始，這告訴瀏覽器這是 HTML5 文件。

常用的 HTML 標籤

標籤 用途 範例
<h1>-<h6> 標題 <h1>主標題</h1>
<p> 段落 <p>這是一個段落</p>
<a> 連結 <a href="url">連結文字</a>
<img> 圖片 <img src="image.jpg" alt="描述">
<div> 區塊容器 <div>內容</div>

什麼是 CSS？

CSS（Cascading Style Sheets）是用來控制 HTML 元素外觀的樣式表語言。它讓我們能夠改變顏色、字體、間距、佈局等視覺效果。

CSS 的三種使用方式

  1. 行內樣式（Inline）：直接在 HTML 標籤中使用 style 屬性
  2. 內部樣式（Internal）：在 HTML 的 <head> 中使用 <style> 標籤
  3. 外部樣式（External）：使用獨立的 .css 檔案

基本的 CSS 語法

/* 選擇器 */
h1 {
    color: blue;           /* 文字顏色 */
    font-size: 24px;      /* 字體大小 */
    text-align: center;   /* 文字對齊 */
}

/* Class 選擇器 */
.highlight {
    background-color: yellow;
    padding: 10px;
}

/* ID 選擇器 */
#header {
    background-color: #333;
    color: white;
}
⚠️ 注意：ID 選擇器（#）的優先級高於 Class 選擇器（.），而行內樣式的優先級最高。

實作練習：創建一個簡單的個人簡介頁面

讓我們結合 HTML 和 CSS 來創建一個簡單的個人簡介頁面：

<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <title>我的個人簡介</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            max-width: 600px;
            margin: 0 auto;
            padding: 20px;
        }
        
        .profile-header {
            text-align: center;
            background-color: #f0f0f0;
            padding: 20px;
            border-radius: 10px;
        }
        
        .profile-image {
            width: 150px;
            height: 150px;
            border-radius: 50%;
        }
        
        .skills {
            display: flex;
            gap: 10px;
            flex-wrap: wrap;
            margin-top: 20px;
        }
        
        .skill-tag {
            background-color: #007bff;
            color: white;
            padding: 5px 15px;
            border-radius: 20px;
        }
    </style>
</head>
<body>
    <div class="profile-header">
        <img src="https://via.placeholder.com/150" alt="個人照片" class="profile-image">
        <h1>張小明</h1>
        <p>前端開發學習者</p>
    </div>
    
    <h2>關於我</h2>
    <p>我是一個熱愛學習的開發者，目前正在學習網頁開發技術。</p>
    
    <h2>技能</h2>
    <div class="skills">
        <span class="skill-tag">HTML</span>
        <span class="skill-tag">CSS</span>
        <span class="skill-tag">JavaScript</span>
    </div>
</body>
</html>
✅ 完成！你已經創建了一個包含個人資訊、照片和技能標籤的簡單網頁。
